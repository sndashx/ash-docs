ASH — Phase 04: In-Game Authoring Tool
========================================

Phase goal: a full in-game editor with paint, select, pick, fill,
entity-placement, warp-to tools. Undo/redo. Cut/copy/paste.
Layer visibility. Save map back to .xp.

Entry criteria: Phase 3 complete.
Exit criteria: editor mode lets you author a new 10x10 map
              entirely in-game, save it, reload it, and have
              entities spawn correctly.

[ENTRY CRITERIA]
  - Phase 3 complete.

[EXIT CRITERIA]
  - Press F7 in EXPLORE mode -> editor opens.
  - All 9 layers switchable.
  - Brush, select, pick, fill, entity, warp tools functional.
  - Undo/redo via ctrl+z / ctrl+y.
  - Cut/copy/paste works.
  - ctrl+s saves the current map.
  - Reloading the saved map yields identical output.

[RISKS]
  - Editor complexity. Mitigation: phased tools, each in own file.
  - Undo stack memory. Mitigation: cap at 1000 commands.
