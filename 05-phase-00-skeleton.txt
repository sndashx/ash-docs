ASH — Phase 00: Project Skeleton
==================================

Phase goal: bootstrap a buildable, testable C++20 project with
CMake, CPM, and a minimal "ASH v0.0.1" truecolor banner.

Entry criteria: nothing; this is the start.
Exit criteria: ./build/ash prints truecolor banner and exits 0.

[ENTRY CRITERIA]
  - None.

[EXIT CRITERIA]
  - cmake -B build succeeds with -Wall -Wextra -Wpedantic.
  - ./build/ash prints banner in truecolor and exits 0.
  - ./build/ash --version prints version and exits 0.
  - ./build/ash --help prints usage and exits 0.
  - ./build/ash --unknown-flag prints error and exits 1.
  - All tests pass (zero tests yet, but Catch2 wired up).
  - Conventional Commits format on git log.

[BLOCKERS]
  - None expected.

[RISKS]
  - Terminal truecolor detection. Mitigation: probe via OSC 4 query
    OR trust modern terminal defaults.
  - CPM cache misses in CI. Mitigation: pre-cache via cpm-cache GitHub
    Action.
