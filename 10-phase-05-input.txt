ASH — Phase 05: Input & Camera
================================

Phase goal: input mode stack, modal UI, configurable keybinds,
mouse support, follow camera, sub-cell smoothing.

Entry criteria: Phase 4 complete.
Exit criteria: a player can walk around a 20x20 map with smooth
              camera follow, mouse picks up items, keybinds are
              rebindable.

[ENTRY CRITERIA]
  - Phase 4 complete.

[EXIT CRITERIA]
  - Mode stack works: EXPLORE > MENU > DIALOGUE etc.
  - Camera follows player smoothly.
  - Mouse clicks pick up items.
  - Keybinds load from ~/.config/ash/keybinds.json.
  - Custom rebinds take effect on restart.

[RISKS]
  - Modal stack correctness. Mitigation: explicit mode transitions
    with assertions.
  - Mouse capture reliability. Mitigation: SGR mode + tested
    terminals.
