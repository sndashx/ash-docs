PHASE 09 — QUEST ENGINE
==========================
Goal: Implement .qst DSL parser, quest journal, objective tracking, branching, simultaneous stages, escorts, investigations, timed objectives, kill/fetch counters, quest givers, hooks into dialogue and triggers.
Reference: appendices/C-quest-format.txt, decisions.txt D-44..D-52.

09.00 — QUEST COMPONENTS (entt)
  09.00.001 — Create src/quest/QuestComponents.hpp
    - struct QuestId { entt::id_type value; std::string key; };
    - struct QuestState { enum Phase { Hidden, Offered, Active, Completed, Failed }; Phase phase; int stage_index; float stage_time; };
    - struct ObjectiveState { std::vector<int> counters; std::set<std::string> flags; std::set<entt::entity> targets; };
    - struct QuestLog { std::vector<entt::entity> active; std::vector<entt::entity> completed; std::vector<entt::entity> failed; std::map<entt::entity, int> unread; };
    - struct QuestGiver { entt::entity npc; std::vector<std::string> quest_keys; };
    - struct EscortState { entt::entity follower; entt::entity destination; float max_distance; bool arrived; };
    - struct InvestigationState { std::vector<entt::entity> clue_entities; int clues_found; int clues_required; };
  09.00.002 — Add to entt registry in World init
    - Register QuestState, ObjectiveState, EscortState, InvestigationState
  09.00.003 — Create src/quest/QuestTypes.hpp
    - enum class StageKind { Objective, Branch, Simultaneous, Escort, Investigate, Timed, Kill, Fetch };
    - struct Stage { std::string id; StageKind kind; std::string text; std::vector<Condition> conditions; };
    - struct QuestDef { std::string key; std::string title; std::string category; std::vector<Stage> stages; std::vector<Stage> branches; int xp_reward; std::vector<ItemRef> item_rewards; std::vector<FactionRep> faction_rewards; };
    - struct Condition { enum Op { HasFlag, FlagEquals, HasItem, Killed, ArrivedAt, TimeOfDay, FactionRank, QuestPhase, VarEquals }; Op op; std::vector<std::string> args; };

  09.02.007 — Add save/load hooks for QuestJournal state
